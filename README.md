# Delhi Air Quality Dashboard

## Project Overview

This project presents an interactive dashboard for analyzing Delhi Air Quality data using Microsoft Excel. The dashboard combines KPI cards, charts, and data storytelling techniques to identify pollution patterns and provide meaningful insights.

## Objectives

- Analyze air quality data from multiple monitoring locations in Delhi.
- Compare pollution levels across locations.
- Identify major pollutant parameters.
- Study hourly pollution trends.
- Present insights through an interactive dashboard.

## Dashboard Components

### KPI Cards
- Total Records: 1,046,223
- Average Pollution Level: 59.99
- Maximum Pollution Level: 1961.70
- Minimum Pollution Level: 0.00
- Total Locations: 4

### Visualizations
1. Average Pollution by Locations (Donut Chart)
2. Average Pollution by Parameters (Bar Chart)
3. Average Pollution by Hours (Line Chart)

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Dashboard Design
- Data Storytelling

## Key Insights

- Anand Vihar recorded the highest average pollution level.
- Pusa showed comparatively lower pollution levels.
- PM10 and PM2.5 are the major contributors to air pollution.
- Pollution levels generally increase during afternoon and evening hours.
- Peak pollution levels are observed between 3 PM and 8 PM.
